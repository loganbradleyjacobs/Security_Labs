class cryptology.ciph_utils.Symbol_Set(symbols)[source]

Bases: object

A class that reconciles the differences in representation between a string of characters passed as an alphabet, and a tuple of a high and low bound of ascii characters.

All utility functions can be passed a Symbol_Set object, initialized as either:

  • SymbolSet((32, 126)): includes all ASCII characters from 32 to 126 inclusive.

  • SymbolSet(“ABCDEF “): includes A through F inclusive and space.

Variables:

  • is_range (bool) – Whether the symbol set is defined by a range

  • low (int) – Lower ASCII bound (if is_range is True)

  • high (int) – Upper ASCII bound (if is_range is True)

  • size (int) – Number of characters in the symbol set

  • allowed (list[str]) – List of allowed characters (if is_range is False)

Parameters:

symbols (tuple[int, int] | str)

__init__(symbols)[source]

Initialize a Symbol_Set with either a range tuple or explicit character string.

Parameters:

symbols (tuple[int, int] | str) – Either a tuple (low, high) defining ASCII code range (inclusive), or a string containing explicit allowed characters

Raises:

  • TypeError – If symbols is not a tuple or string, or if a list is provided

  • ValueError – If tuple range is invalid (low > high)

Example

>>> Symbol_Set((32, 126))  # Printable ASCII
>>> Symbol_Set("ABCDEF ")   # A-F and space

Convert a string to a list of integers, validating against a Symbol_Set.

By default, characters are converted to their ASCII values. If start_index is provided, characters are converted to their index within the Symbol_Set, offset by start_index.

Parameters:

  • message (str) – String to convert

  • symbols (Symbol_Set) – Symbol_Set defining valid characters

  • start_index (int | None) –

    • None (default): return ASCII values via ord(c)

    • int k: return symbol indices starting at k

Returns:

  • ASCII codes if start_index is None

  • Symbol indices (offset by start_index) otherwise

Return type:

list[int]

Examples

>>> Utils.ord_str("AB C", Symbol_Set("ABC "))
[65, 66, 32, 67]

>>> Utils.ord_str("abc", LOWER, start_index=1)
[1, 2, 3]

>>> Utils.ord_str("ABC", UPPER, start_index=0)
[0, 1, 2]

Parameters:

  • keyword (str) – The keyword used to determine shift values for each character position

  • encrypt (bool) – Boolean flag indicating whether to encrypt (True) or decrypt (False) the message

  • symbols (Symbol_Set) – Symbol_Set defining the valid character range. If None, uses default printable ASCII (32-126)

Returns:

The resulting ciphertext (if encrypting) or plaintext (if decrypting)

Return type:

str

Example

>>> vigenere_cipher("HELLO", "KEY", True)
'RIJVS'
>>> vigenere_cipher("RIJVS", "KEY", False)
'HELLO'

Note

The keyword is case-sensitive and uses modular arithmetic for character shifts. Each character in the keyword provides a shift value based on its position in the symbol set.

Gets cross correlation between two similar sets, returns the cross correlation for every shift. Intuition: if dict2 slides past dict1, at what shift do they match the best?

Parameters:

  • dict1 (dict[str, float]) – First frequency dictionary (e.g., observed frequencies)

  • dict2 (dict[str, float]) – Second frequency dictionary (e.g., expected frequencies or another observed set)

  • symbols (Symbol_Set) – Symbol_Set defining the character order (defaults to printable ASCII)

Returns:

Cross-correlation values for each possible shift (0 to n-1 where n is symbol set size)

Return type:

list[float]

Example

>>> dict1 = {'A': 0.5, 'B': 0.3, 'C': 0.2}
>>> dict2 = {'A': 0.2, 'B': 0.5, 'C': 0.3}
>>> cross_correlation(dict1, dict2, Symbol_Set("ABC"))
[0.38, 0.31, 0.31]  # Highest value at shift 0 indicates best match at no shift
cryptology.symmetric.cryptanalysis.get_caesar_shift(enc_message, expected_dist, symbols=None)[source]

Gets the likely shift used to originally encrypt a caesar cipher.

Parameters:

  • enc_message (str) – Encrypted ciphertext to analyze

  • expected_dist (dict[str, float]) – Dictionary of expected character frequencies for the language

  • symbols (Symbol_Set) – Symbol_Set defining valid characters (defaults to printable ASCII)

Returns:

Most likely Caesar cipher shift value (0 to n-1 where n is symbol set size)

Return type:

int

Example

>>> english_dist = {'A': 0.082, 'B': 0.015, ...}  # English letter frequencies
>>> get_caesar_shift("XYZ", english_dist, Symbol_Set("ABC...XYZ"))
3  # Indicates likely shift of 3 (X=A+3, Y=B+3, Z=C+3)

Note

This function will only work with the ‘upper_alphabet_with_space symbol set = Symbol_Set(“ABCEDFGHIJKLMNOPQRSTUVWXYZ “) Any difference in size between the symbol sets will muddle the modular arithmetic.
