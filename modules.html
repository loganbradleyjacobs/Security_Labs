class cryptology.ciph_utils.Symbol_Set(symbols)[source]

Bases: object

A class that reconciles the differences in representation between a string of characters passed as an alphabet, and a tuple of a high and low bound of ascii characters. All utility functions can be passed a Symbol_Set object, initialized as either: SymbolSet((32, 126)): includes all ASCII characters from 32 to 126 inclusive. Or, SymbolSet(“ABCDEF “): includes A through F inclusive and space.
